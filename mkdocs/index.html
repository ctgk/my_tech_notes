    日本語検索

    日本語検索に対応させるにはmkdocs.ymlに以下を追加

    extra:
    search:
        language: 'ja'

    GitHub連携

    MkDocsで作ったサイトをgithubで公開する手順

    1. githubで空のレポジトリを作成
      • 仮にmydocsとする
    2. $ git clone mydocs
    3. $ mkdocs new mydocs
    4. $ mkdocs gh-deploy
    5. githubのrepositoryからsettings→GitHub Pages→Source→gh-pages branch
    6. アクセスできるようになるまで少し待つ
    7. サイトにアクセス
