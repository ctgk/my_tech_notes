    インストール

    Linux

    Linuxの場合はgit, pyenv, minicondaを使っていく

    1. pyenvをダウンロード

      1
      git clone https://github.com/pyenv/pyenv.git ~/.pyenv

    2. githubのページからリリースを確認して、checkout

      1
      git checkout v1.2.12

    3. .bashrcを編集

      1
2
3
4
      $ vim ~/.bashrc
export PYENV_ROOT="$HOME/.pyenv"
export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init -)"

    4. .bashrcを更新

      1
      source ~/.bashrc

    5. 好みでpyenv_aliasをインストール

      1
      git clone https://github.com/s1341/pyenv-alias.git $(pyenv root)/plugins/pyenv-alias

    6. インストールできる環境を確認する

    7. インストールしたい環境をインストール

      Note

      miniconda3-latestをインストールするとインストール後に確認するときもminiconda3-latestと表示される。 versionを確認できるようにしたかったので、これはインストールしない。

    8. インストールできたか確認

      1
2
3
      $ pyenv versions
* system (set by ~/.pyenv/version)
miniconda3-4.3.30

    9. systemのpythonからminicondaのpythonに切り替える

      1
      pyenv global miniconda3-4.3.30

    10. 切り替わったか確認

      1
2
      $ python --version
