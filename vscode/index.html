    SSH FS

    2019/4/30にMicrosoft自ら「Remote Development」という拡張機能を発表している。 まだInsider Previewだが、正式リリースされたらそっちのほうがいいかも

    {
    "sshfs.configs": [
        {
            "host": "<ip address>",
            "name": "raspberrypi",
            "password": true,
            "port": 22,
            "root": "/home/pi/",
            "username": "pi"
        }
    ]
}
